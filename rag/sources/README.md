# RAG Source Notes

This directory stores lightweight source-grounding notes for the RAG knowledge library.

The notes should record official or public reference locations, local rule metadata, and distilled usage boundaries for session-level official code judgment. They must not store full webpages, full rule bodies, current test-set answers, expected labels, MTA answers PDF concrete answers, raw private PCAP paths, or secrets.

When a source note supports an older event-card concept, keep the note but mark the usage as legacy or intermediate. Final competition outputs must use official stage / technique codes and CSV submission files.
